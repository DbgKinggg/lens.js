# Installation

You can install Lens.js by running the below command inside your project directory:

```sh
npm install lens-protocol
```

Or with `yarn`:

```sh
yarn add lens-protocol
```

Once the installation is complete, you can import the library in your project:

```js
import { Lens } from 'lens-protocol';
```

> Please note that this library is at its very early stage,some APIs are not yet supported and there might be some breaking changes. If you find any, please add them on
> [GitHub](https://github.com/suhailkakar/lens.js)
